#Intro
this element represents a one dimensional spring element.

#Special cases
if start and end nodes of spring have same location (zero length spring) then stiffness matrix will be like this:

	  Dx1  Dx2
Dx1   +K    -K
Dx2   -K   +K

	  Dy1  Dy2
Dy1   +K    -K
Dy2   -K   +K

	  Dz1  Dz2
Dz1   +K    -K
Dz2   -K   +K
